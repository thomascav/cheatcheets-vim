Quelques commandes utiles de Vim

Première chose avec Vim, une fois ouvert, nous avons envie de quitter !

: wq permet de quitter Vim
: wq permet de quitter en sauvegardant
: wq nom de dossier permet d'enregistrer dans le dossier spécifié et de quitter

Taper du texte dans Vim

i permet d'insérer du texte
a permet d'ajouter du texte

Supprimer du texte

x permet de supprimer une lettre après le curseur
dw permet de supprimer le mot (curseur au début du mot)
d$ permet de supprimer du texte jusqu'à la fin de la ligne (après le mot où se trouve le curseur)
